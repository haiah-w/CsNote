# ASCII

包含：英文、符号；

- 使用<mark>1个字节</mark>存储一个字符

- <mark>首位是0</mark>

- 共存储<mark>128个字符</mark>

# GBK

汉字编码字符集，包含2w多个汉字，兼容ASCII字符集；

- 使用2个字节存放汉字；第一个字节的第一位必须是1；

- 1个字节存放ASCII码；
